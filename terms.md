### Supported Functionalities

#### Functionalities

- **A01.** Dynamic VAT calculation based on destination rules for international transactions.
- **A02.** Definition of multiple ports, routes, and legs for VAT determination.
- **A03.** VAT adjustment via timetables, manual leg changes, or automated sources such as GPS inputs.
- **A04.** Updating VAT Business Posting Groups in LS Central POS transactions.

#### Language Features

- **B01.** English user interface.
- **B02.** Optional translations depending on project scope.

#### Technical Features

- **C01.** Developed according to Microsoft AL/BC/LS standards.
- **C02.** Installation requires publishing extensions and configuration.

#### Exclusions

- **D01.** Local statutory requirements unrelated to dynamic VAT.
- **D02.** External system integrations unless specified.
- **D03.** Reports not related to VAT logic.
- **D04.** Any function not explicitly listed.
